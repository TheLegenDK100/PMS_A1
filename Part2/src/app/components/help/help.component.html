Help, FAQs, and Troubleshooting

FAQs

Q: Why does data disappear after closing browser?

A: The app is session-based with no server/database persistence.

Q: Why cannot I use an existing Item ID?

A: Item ID must be unique and can only be saved once.

Q: How do I update an item?

A: Go to Manage Items, load or enter an existing item name, edit details, then use Update by Name.

Troubleshooting

  • If add/update fails, check required fields and numeric values.
  • If delete does not run, ensure the item name exists and confirm deletion.
  • If search returns nothing, try partial names or show all items first.
